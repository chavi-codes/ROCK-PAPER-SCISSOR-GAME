✊✋✌️ Rock Paper Scissors Game

A fun and interactive Rock Paper Scissors web application where users can play against the computer. The game features real-time results, score tracking, and a clean, responsive user interface.

🚀 Features

🤖 Play Against Computer – Randomized computer choices

⚡ Real-Time Results – Instant win/lose/draw outcome

📊 Score Tracking – Keeps track of player and computer scores

🔄 Restart Option – Reset the game anytime

📱 Responsive Design – Works smoothly on all devices

🎨 Interactive UI – Simple, clean, and engaging design

🛠️ Technologies Used

HTML5 – Structure of the application

CSS3 – Styling and layout

JavaScript – Game logic and functionality

🎯 How to Play

1.Choose Rock, Paper, or Scissors

2.The computer randomly selects its move

3.The winner is decided based on standard rules:

-Rock beats Scissors

-Scissors beats Paper

-Paper beats Rock

4.Scores are updated after each round
